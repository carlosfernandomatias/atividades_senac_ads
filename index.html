Dashboard SENAC RN

Atividade 1

Este projeto é uma aplicação simples em HTML, CSS e JavaScript que permite ao usuário verificar se uma cidade pertence a um estado específico no Brasil, utilizando a API do IBGE.

Atividade 2

Este projeto é uma aplicação simples em HTML, CSS e JavaScript que permite criar uma lista de itens organizados por categorias. O usuário pode adicionar itens a categorias específicas, e cada categoria exibe seus respectivos itens em uma lista ordenada.

Atividade 3

Este projeto é uma aplicação simples que permite ao usuário digitar uma palavra e um caractere específico para verificar a quantidade de vogais e quantas vezes o caractere aparece na palavra.

Atividade 4

Esta é uma aplicação de Calculadora Simples desenvolvida com HTML, CSS e JavaScript puro. O objetivo é permitir que o usuário selecione uma operação matemática e veja o resultado.

Atividade 5

Este é um jogo simples de adivinhação desenvolvido em HTML, CSS e JavaScript. O objetivo do jogo é adivinhar um número gerado aleatoriamente entre 1 e 100. O usuário insere um número, e o sistema informa se o número é maior ou menor do que o valor gerado, até que o usuário adivinhe corretamente.

Atividade 6

Este é um simples sistema de cadastro de usuários, onde você pode adicionar, editar ou excluir registros de usuários em uma tabela. Os dados cadastrados incluem: Nome, Telefone, E-mail, Data de Nascimento, Profissão e Endereço.

Atividade 7

Este é um simples sistema de cadastro de usuários, onde você pode adicionar, editar ou excluir registros de usuários em uma tabela. Os dados cadastrados incluem: Nome, Telefone, E-mail, Data de Nascimento, Profissão e Endereço.
